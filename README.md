# Інформаційна система оренди автомобілів (Car Rental DSS)

Прототип інформаційної системи для підбору та оренди автомобілів з використанням методу аналізу ієрархій (AHP).

## Інструкція для розробників (Onboarding)

Цей посібник допоможе вам розгорнути проєкт на свіжій ОС для подальшої розробки.

### 1. Необхідні залежності та ПЗ

Перед початком роботи переконайтеся, що на вашому комп'ютері встановлено:

- **Git** (для контролю версій)
- **Python 3.10+** (обов'язково з додаванням у PATH)
- **Visual Studio Code** (або інший зручний редактор)

### 2. Налаштування середовища розробки

Склонуйте репозиторій на свій комп'ютер:

```bash
git clone [https://github.com/stask19/bachelor-landing.git](https://github.com/stask19/bachelor-landing.git)
cd bachelor-landing
```
