# Процедура оновлення системи

Для безпечного оновлення інформаційної системи до нової версії виконайте наступні кроки:

1. Підключіться до сервера по SSH.
2. Перейдіть до директорії проєкту: `cd /var/www/car-rental-dss`.
3. Отримайте останні зміни з репозиторію: `git pull origin main`.
4. Якщо були додані нові бібліотеки (наприклад, для складніших математичних розрахунків AHP), активуйте віртуальне середовище та виконайте: `pip install -r requirements.txt`.
5. Перезапустіть службу бекенду: `sudo systemctl restart car-rental-backend`.
6. Очистіть кеш Nginx (за необхідності) та перевірте працездатність сайту.
