# Інструкція з розгортання (Production Deployment)

Цей документ описує базовий процес розгортання Інформаційної системи з модулем AHP на Linux-сервері.

## Вимоги до сервера

- ОС: Ubuntu 22.04 LTS
- Встановлені пакети: Python 3.10+, Nginx, Git, Gunicorn.

## Кроки розгортання:

1. **Підготовка сервера:** Оновлення пакетів `sudo apt update && sudo apt upgrade`.
2. **Клонування коду:** Клонування репозиторію у директорію `/var/www/car-rental-dss`.
3. **Налаштування Backend:** \* Створення віртуального середовища (`python3 -m venv venv`) та встановлення залежностей.
   - Налаштування Gunicorn як WSGI-сервера для постійної роботи Flask-додатка.
   - Створення systemd-сервісу для автоматичного перезапуску бекенду після збоїв.
4. **Налаштування Web-сервера:**
   - Конфігурація Nginx для роздачі статики (HTML/JS/CSS).
   - Налаштування `proxy_pass` у Nginx для перенаправлення API-запитів (`/api/`) на локальний порт Gunicorn (5000).
5. **Запуск:** Перезавантаження Nginx `sudo systemctl restart nginx`.
