# Документація з налаштування лінтера

## 1. Обраний інструмент

Для даного проєкту було обрано **Prettier** (як розширення для VS Code) та **HTMLHint**.
**Причина вибору:** Prettier забезпечує автоматичне форматування коду згідно з заданими правилами, що підвищує читабельність та уніфікує стиль у файлах HTML та CSS. HTMLHint дозволяє виявляти помилки у структурі тегів.

## 2. Базові правила та їх пояснення

Налаштування зберігаються у файлі `.prettierrc`:

- `tabWidth: 2` — встановлює відступ у два пробіли (стандарт для веб-розробки).
- `semi: true` — автоматично додає крапки з комою в CSS для запобігання помилкам.
- `singleQuote: false` — використовує подвійні лапки для HTML-атрибутів.
- `printWidth: 100` — обмежує довжину рядка для зручності перегляду коду.

## 3. Інструкція з запуску

1. Встановити розширення "Prettier - Code formatter" у VS Code.
2. Використовувати комбінацію клавіш `Shift + Alt + F` для примусового форматування файлу.
3. Або увімкнути опцію "Format on Save" у налаштуваннях редактора для автоматичного виправлення при кожному збереженні.
